Test

It is good to test AngularJS applications. The Todo application implements only a couple of tests but they are always nice to see running.

Run the test task from the odo project directory.

angularity test

This will generate an /app-test directory and you should see a console output indicating all tests passing.

Build

Run the build task from the Todo project directory.

angularity build

This will perform a one-time build in an /app-build directory. For development however, you will find more use in the watch task.

Watch

The watch task is best for normal development.

angularity watch

It is similar to the build task. However it also watches your project for changes and serves the project on a localhost port. For the Todo application this is port 55555.

Navigate your browser to http://localhost:55555.

Release

The release task creates a set of deployment of files.

angularity release

This will generate an /app-release directory with a maximally flat set of files for deployment.
